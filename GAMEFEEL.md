# 게임 필 개선 계획 (뱀파이어 서바이버식 타격감·물량·파워 판타지)

"화려하고, 좀비가 많아 긴장감 있고, 시원시원한" 플레이를 만들기 위한 개선 메뉴.
세 축 — **타격감(Juice) · 물량(Swarm) · 파워 판타지(Power fantasy)** — 로 정리한다.

## ✅ 이번에 적용 (안전·고임팩트 배치)
- [x] **화면 흔들림(screen shake)** — `Events.shake()` → 플레이어 카메라가 감쇠 오프셋 적용.
      피격(7)·보스 폭사(11)·자폭(5)·포격 착탄(4). 멀미 방지 상한(13px)·큰 이벤트에만.
- [x] **히트스톱(hit-stop)** — 보스 폭사 순간 시간을 아주 짧게 멈춰 "한 방의 무게"를 준다
      (시간배율 무시 타이머로 확실히 복구).
- [x] **물량 상향** — 동시 좀비 상한 max_z 32~120 → 42~170. 좀비끼리 물리 충돌 제거 후
      생긴 여유로 화면을 더 빽빽하게(스웜 압박).

## ⬜ 다음 후보 (임팩트 순)

### A. 타격감 (Juice)
- [ ] **데미지 숫자(floating damage)** — 명중 시 숫자가 튀어오르며 사라짐. 크리티컬은 크게/노랗게.
      ※ 대량 히트 시 노이즈·부하 → "큰 피해/보스/크리만 표시" 또는 풀+스로틀로 제어.
- [ ] **사망 연출 강화(gibs/blood)** — 현재 원형 버스트 1개 → 피 튀김 스플랫 + 파편 몇 개 + 흰 섬광.
      (FXBurst 확장/전용 파티클, 풀링 유지).
- [ ] **넉백(knockback)** — 피격 좀비가 살짝 밀려남 → 총알이 "박히는" 느낌.
- [ ] **크리티컬 히트** — 확률 크리 + 크리 전용 이펙트/사운드/흔들림.
- [ ] **카메라 킥/줌 펀치** — 발사·폭발 시 미세 줌 인/아웃.

### B. 물량·긴장감 (Swarm)
- [ ] **엘리트 팩/스웜 이벤트** — 주기적으로 한 종이 떼로 몰려오는 웨이브 내 미니 이벤트.
- [ ] **연속 스폰 압박 곡선** — 6웨이브 이후 밀도도 점증(현재는 HP·이속만 압박).
- [ ] **위험 근접 경고** — 사방에서 포위될 때 가장자리 경고 연출(현재 저체력 비네팅 확장).
- [ ] **성능 가드** — 대량 동시 시 FX 동시 수 상한·좀비 원경 컬링(프레임 방어).

### C. 파워 판타지 (Power fantasy)
- [ ] **레벨업/무기 선택 루프** — 뱀서 핵심. 경험치 젬 수집 → 레벨업 시 3택 강화 카드.
      (현재는 웨이브 간 상점. 인게임 실시간 성장으로 바꾸면 몰입·시원함↑ — 큰 작업).
- [ ] **화면 정리기(스크린 클리어)** — 폭탄/필살기로 잡몹 일소 + 대형 연출.
- [ ] **연쇄/전이 효과** — 번개 체인, 폭발 연쇄로 "한 번에 우수수" 쾌감.
- [ ] **킬 스트릭 피드백** — 초당 처치 수에 따라 배경 채도/음악 레이어/콤보 텍스트 상승.
- [ ] **오토 조준 강화 시각화** — 다중 표적 동시 사격 라인 등 "쏟아붓는" 화면.

## 권장 다음 순서
1. **사망 연출 강화 + 넉백**(A) — 물량과 결합해 "우수수 터지는" 쾌감을 즉시 키움, 저위험.
2. **엘리트 팩/스웜 이벤트**(B) — 긴장감의 리듬을 만든다.
3. **데미지 숫자**(A, 스로틀 적용) — 피드백 명료화.
4. (대형) **레벨업/무기 선택 루프**(C) — 게임 정체성을 뱀서식으로 굳히는 큰 결정.

> 밸런스에 영향 주는 항목(물량·레벨업)은 실플레이 후 수치 조정(BALANCE)과 함께 간다.
