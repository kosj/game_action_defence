# 뱀파이어 서바이버식 성장·아이템 시스템 제안

현재도 "코인=경험치 → 레벨업 3택 카드"로 뱀서의 뼈대는 있다. 진짜 뱀서의 재미는
**여러 무기를 슬롯에 모으고 → 레벨업 → 조합으로 진화**시키는 빌드 구성에 있다.
아래는 그 방향으로 단계별로 확장하는 제안이다(각 단계 독립 배포 가능).

## 현재 vs 뱀서
| 요소 | 현재 | 뱀서 |
|---|---|---|
| 성장 | 레벨업 3택(스탯 강화) | 레벨업 3택(무기/패시브 **아이템**) |
| 무기 | 자동총 1 + 오브 + 낙뢰(스탯 업글) | **무기 슬롯 6개** 각자 Lv1~8로 거동 변화 |
| 패시브 | 크리·재생·자석 등(스탯) | **패시브 슬롯 6개**(별도) |
| 진화 | 없음 | 무기 만렙 + 특정 패시브 → **진화 무기** |
| 획득 | 레벨업만 | 레벨업 + **상자**(엘리트/보스 드랍) |
| 메타 | 없음(골드 사용처 없음) | **영구 강화**(런 간 골드 소비) + 캐릭터 해금 |
| 런 구조 | 무한 웨이브 | 생존 시간 + 시간대별 보스 |

---

## 단계별 제안

### A. 무기/패시브 아이템 프레임 (핵심 리팩터) ⭐
- **아이템 카탈로그**(데이터 주도): `WEAPONS`(자동총·오브·낙뢰 + 신규)와 `PASSIVES`(크리·재생·자석·이속·공속·최대HP·탐욕 등).
- **인벤토리 슬롯**: 무기 최대 6 + 패시브 최대 6. 레벨업 카드는 "새 아이템 획득" 또는 "보유 아이템 레벨업"을 제시.
- 각 무기는 **Lv1~8**로 수치/거동이 바뀐다(예: 자동총 → 관통↑·투사체↑).
- 기존 자동총/오브/낙뢰/현재 업글을 이 구조로 이관(자동총=무기, 크리/재생 등=패시브).
- **재사용**: 기존 Bullet·Orb·Lightning·EnemyBullet·FXBurst 인프라를 무기 구현에 그대로 활용.

### B. 신규 무기 다양화
뱀서 손맛의 핵심. 기존 FX 인프라로 저비용 구현 가능한 것부터:
- **채찍/부채꼴 근접**(전방 아크 타격), **부메랑/회귀 투사체**, **유도 미사일**,
  **오라(마늘)**(주변 지속 광역), **지뢰/설치물**, **빙결 노바**(광역 슬로우), **체인 라이트닝 강화**.

### C. 무기 진화 (뱀서 시그니처)
- **만렙 무기 + 특정 패시브** 보유 시, 레벨업/상자에서 **진화 무기** 등장.
  - 예: 자동총(만렙)+크리 → **레일건**(관통·크리 특화) / 오브(만렙)+오브속도 → **톱날 폭풍**.
- "무엇을 만렙 찍고 무엇과 조합할지" 빌드 목표를 만든다.

### D. 픽업 & 상자
- **경험치 젬 등급화**(소/중/대 — 강적일수록 큰 젬).
- **상자**: 엘리트/보스 처치 시 드랍 → 레벨업 1회 또는 진화 트리거.
- **소모 픽업**: 폭탄(화면 정리)·빙결·자석·회복. (현재 자석/무기 픽업 확장)

### E. 메타 성장 (런 간 영구) — 골드 재활용 ⭐
- 상점을 없애며 골드가 놀고 있다 → **메인 메뉴 "PowerUp" 화면**에서 골드로 **영구 강화** 구매.
  - 시작 데미지 +%, 최대 체력 +, 부활 +1, 행운(진화·상자 확률) +, 시작 골드/경험치 + 등.
- **캐릭터 해금**: 서로 다른 시작 무기·패시브·스탯(플레이 변주). 골드/업적으로 해금.
- 디스크 저장(SaveManager 확장). "죽어도 조금씩 강해진다"는 리텐션 루프.

### F. 런 구조(선택)
- **생존 타이머**(예: 15~20분) + 시간대별 스폰 격화 + 정해진 시각 보스 등장.
- 끝까지 생존 시 최종 보스("사신") — 클리어 목표 명확화(GOALS 와 연결).
- 현재 무한 웨이브와 택1 또는 혼합("웨이브=난이도 시계").

## 권장 순서
1. **A 무기/패시브 프레임 + 슬롯**(핵심 리팩터) — 이후 모든 것의 토대. 기존 업글을 이관.
2. **B 신규 무기 2~3종** — 빌드 다양성 즉시 체감(저비용부터: 오라·부메랑·빙결노바).
3. **E 메타 성장(영구 강화)** — 놀고 있는 골드에 의미 부여 + 리텐션.
4. **C 진화** — 빌드의 정점(엔드게임 쾌감).
5. **D 상자/픽업 등급화** + **F 런 구조** — 완성도.

> A는 큰 리팩터라 신중히(현재 잘 도는 오브/낙뢰/자동총을 무손실 이관). 원하시면 A부터 착수하되,
> 먼저 "무기 6 + 패시브 6 슬롯 / 진화 유무 / 캐릭터 해금 유무" 범위를 확정하고 진행한다.

## 저비용 vs 고비용
- **저비용(기존 인프라 재사용)**: 신규 무기(B) 다수, 소모 픽업(D 일부), 메타 영구강화(E, UI만).
- **고비용(신규 설계)**: 무기/패시브 슬롯 프레임(A), 진화 규칙(C), 캐릭터 시스템(E), 런 타이머(F).
